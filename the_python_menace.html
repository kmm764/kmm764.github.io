Inspiration
This is the first software development for me. Working in a group of three, we all agreed with my idea of making a top down 2D shooter. This game is inspired by a top down shooting game I played in my childhood, where the player is controlling a character throughout the world war ages.

Aim of the Game
In this game, the hero character is controlled by the player. The main mission of the hero is to collect the five backpacks , which are the supplies to survive and then go to the platform that will lead him to the next level if he already has all backpacks.

How We built it
The game is developed with PyCharm using PyGame, designed for PC.

Challenges we ran into
It turns out that rotation of the player is one of the hardest element to impletement, especially when it comes to collision with different objects such as walls and zombies. Surprisingly, it is quite challenging to impletement a Highest score board, storing player information on the PC.

Accomplishments
Project was awarded game of the year, placed first in the MSc computer science course, achieving 95% for the module.
