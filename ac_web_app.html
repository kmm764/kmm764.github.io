Description
Learning Information Web app is a system with a purpose of displaying user's acedemic information. HTML/CSS is used for frontend, php for backend and SQL database is used. The Database web app used is in third normal form. Where the Database should have no functional and transitive dependencies.

Description of Users
University username and password is needed, as well as basic knowledge of using web browsers, and the basic knowledge of website navigation. The user should be a former or current university student. The user also should be capable of using mouse and keyboard, have basic knowledge of mouse usage and typing. Additionally, user must be abled to have access to device such as mobile or PC in order to have access to the website.

Description of Prototype
University students should be allowed to access the bluecastle system. Where the system consist if their modules for the year, and module taken in the past with the correct credits and semester of each module shown. Award, Progression, Marks are also available in the system after the user log in with the correct username and password.
