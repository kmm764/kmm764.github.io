Game Description
This game is in development and in open Beta on Google Playstore and close to be published on Steam. Furture details, pictures and trailer will be updated ASAP. This is a single player game made with Unity, the goal of the game is to help the character survive the endless canyon, collecting orbs on the way to cast abilities. While avoiding hazards such as spiky platforms and fireballs, and defeating bosses.

Game Rules
The player must not go past the viewport’s top and bottom boundary, or the game will be over. The player can jump to different platforms and stay alive within the viewport. The player will lose HP if in contact with the spike platform or get hit by a missles. There are boundaries on the right and left side as well, restriction the player from entering. The boundaries are same as white walls on the background.

Game Flow
The game encourages the player to survive as long as possible, as there are leader boards. The game is easy to learn but hard to master. While all the game objects are randomly generated, with Bosses and wands atacking the player. The game relies on the player’s reaction time.

How I built it
The game is developed with Unity, designed for PC and Andriod.
